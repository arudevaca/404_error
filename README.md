
# 404 Error Page - Aprendiendo Código

Este es un proyecto de práctica donde he desarrollado una página de error 404 personalizada. El objetivo principal fue dominar el uso de **Flexbox** y la implementación de **Diseño Responsivo (Mobile First)** para que la página se vea perfecta tanto en computadoras como en dispositivos móviles.

## Características

* **Diseño Responsivo:** Adaptado para móviles, tablets y escritorio mediante Media Queries.
* **Interfaz Limpia:** Uso de tipografía monospace para un estilo técnico y moderno.
* **Interactividad:** Botones con efectos hover y navegación funcional.
* **Optimización:** Estructura HTML5 semántica y CSS3 eficiente.

## Tecnologías utilizadas

* **HTML5:** Estructura del sitio.
* **CSS3:** Estilos, Flexbox y Media Queries.
* **Git & GitHub:** Control de versiones y despliegue.

## Vista Previa (Responsive)

El sitio cambia su estructura dependiendo del tamaño de la pantalla:
- **Móvil/Tablet:** Los elementos se apilan verticalmente para facilitar la lectura.
- **PC:** Los elementos se distribuyen horizontalmente aprovechando el ancho de la pantalla.
